# Vlaams woordenboek - Documentatie 

Welkom op de repository van de documentatie voor het Vlaams Woordenboek. Dit platform biedt een gestructureerd overzicht van de documenttie rond de werking, 
ontwikkeling en gebruik van het Vlaams Woordenboek. 

## Over dit portaal 

Dit documentatieportaal is gebouwd met [Nextra](https://nextra.site/), een lichte en snelle statische site generator op basis van React en Typescript. 
Het dient als centrale hub voor ontwikkelaars, redacteus en gebruikers die meer willen weten over de functionaliteiten en bijdragen van het Vlaams Woordenboek. 

## Inhoud 

- **Gebruikershandleiding:** Hoe je het woordenboek kunt doorzoeken, beheren en of gebruiken.
- **Bijdragen:** Hoe je nieuwe woorden en definities toevoegt of bewerkt. 
- **Technische documentatie:** Voor ontwikkelaars die willen bijdragen aan de code. 
- **Veelgestelde vragen (FAQ):** Antwoorden op de meest voorkomende vragen. 
- **Community richtlijnen:** Hoe je respectvol en constructief kunt bijdragen. 

## Deployment 

De productie build wordt automatisch uitgevoerd via een Githuib Wokflow en gehost op **Github Pages**. Elke wijziging in de `main` brach triggert een nieuwe build en publicatie. 

Voor handmatige build kun je het volgende commando uitvoeren: 

```bash 
npm run build
```
